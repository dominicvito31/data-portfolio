# Data Engineering & Analytics Portfolio

This repository contains a collection of end-to-end data engineering and analytics projects developed during my data analytics bootcamp. The projects demonstrate practical implementation of data pipelines, dimensional data modeling, workflow orchestration, statistical analysis, and business intelligence visualization.

## Tech Stack

Python • PySpark • SQL • PostgreSQL • MongoDB • Apache Airflow • Docker • Great Expectations • Tableau • Pandas • BeautifulSoup

---

# Projects Overview

## 1. Supply Chain Data Warehouse & Automated ETL Pipeline

End-to-end data warehouse system built using PySpark and Apache Airflow to process a global supply chain dataset.

**Key Features**

* Designed a Snowflake schema data warehouse
* Implemented ETL pipeline using PySpark
* Automated workflows with Airflow DAG scheduling
* Loaded warehouse tables into PostgreSQL cloud database
* Generated surrogate keys for dimensional modeling

**Technologies**

PySpark, Airflow, PostgreSQL, Docker, Python

Repository:
https://github.com/dominicvito31/supply-chain-data-warehouse
